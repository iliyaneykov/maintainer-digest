# Roadmap

## 0.1

- Deterministic issue and PR scoring.
- Markdown maintainer digest.
- Release-note drafting from PR metadata.
- Label suggestions.
- Codex-ready prompt generation.
- GitHub Actions workflow for scheduled digest artifacts.

## 0.2

- Optional OpenAI summarizer with dry-run and redaction options.
- Config file for repository-specific scoring rules.
- Comment-safe mode for GitHub Actions.
- Better handling of stale items, linked issues, and milestone data.

## 0.3

- MCP adapter for agent-native maintainer workflows.
- Web dashboard generated from static Markdown/JSON.
- Maintainer handoff report for vacations or release freezes.

## Non-goals

- Replacing human maintainer judgment.
- Automatically merging code.
- Posting AI-generated comments without explicit maintainer approval.
